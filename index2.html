Text & Font Properties

Typography controls how text looks and reads. Below are the main properties with examples. Hover the example box to see a short description (native tooltip).

color
color: #ef4444;
/* sets text color */ p { color: #ef4444; }
Details

Accepts color keywords, hex, rgb/rgba, hsl/hsla. Use contrast-aware colors for accessibility.

font-family
font-family: Georgia, serif;
p { font-family: 'Georgia', serif; }
Details

Provide fallbacks and generic families (serif, sans-serif, monospace). Web fonts (Google Fonts) can be loaded with @import or .

font-size
font-size: 18px;
p { font-size: 18px; /* px, em, rem, % */ }
Details

Prefer responsive units: rem (root-based) or em (relative to parent). Use clamp() for fluid sizes.

font-weight
font-weight: 700;
p { font-weight: 700; /* normal, bold, 100-900 */ }
Details

Numeric weights depend on the font's available weights. Use variable fonts for more granular control.

text-align
text-align: center;
p { text-align: center; /* left, right, center, justify */ }
Details

Useful for headings and layout. Use text-justify for long passages when necessary.

text-decoration
underline (accent)
a { text-decoration: underline orange 3px; text-underline-offset: 6px; }
Details

Modern text-decoration accepts color and thickness. text-underline-offset controls gap between text and line.

letter-spacing & word-spacing
Spaced text
p { letter-spacing: 1.5px; word-spacing: 4px; }
Details

Useful for headings and improving readability when used sparingly.

line-height
line-height: 1.8
p { line-height: 1.8; /* unitless recommended */ }
Details

Use unitless values (e.g., 1.5) for scalable line spacing that multiplies by font-size.

text-shadow
text-shadow
p { text-shadow: 2px 2px 6px rgba(0,0,0,0.6); }
Details

Multiple shadows allowed (comma-separated). Useful for subtle depth; avoid overuse for accessibility.

Background Properties

Background controls let you layer colors, images, gradients, and control how they behave.

background-color
background-color: #60a5fa;
div { background-color: #60a5fa; }
Details

Use contrast-aware colors and CSS variables for themes.

background-image (gradients)
linear-gradient
div { background: linear-gradient(to right, #f97316, #f59e0b); }
Details

Also radial-gradient and conic-gradient available. Gradients are images (no HTTP request).

background-size & position
size: cover; position: center;
div { background-size: cover; background-position: center; }
Details

cover fills the box (may crop), contain fits the whole image. Use position with two values (x y).

background-repeat & attachment
repeating pattern
div { background-repeat: no-repeat; background-attachment: fixed; }
Details

background-attachment: fixed can create parallax-like effects. Beware performance on mobile.

background-blend-mode
mix-blend-mode: multiply
div { background: linear-gradient(45deg,#f87171, #60a5fa); background-blend-mode: multiply; }
Details

blend modes behave like Photoshop blending; useful for layered backgrounds and overlays.

Box Model

Everything visual in CSS is a box. Understand sizing, spacing, borders, and the box-sizing model.

width / height
width:180px; height:84px;
div { width: 180px; height: 84px; }
Details

Prefer flexible units (%, vw, rem) for responsive UIs.

box-sizing
border-box
* { box-sizing: border-box; }
Details

Set this globally to make sizing intuitive (elements include padding & border in their width/height).

padding & margin
padding:18px; margin:8px;
div { padding: 18px; margin: 8px; }
Details

Shorthand: padding: top right bottom left; Use logical properties (padding-inline, padding-block) for internationalization.

border & border-radius
border:3px solid; border-radius:12px
div { border: 3px solid #eab308; border-radius: 12px; }
Details

Use border-radius for pill buttons and shapes. Accepts 1-4 values or / for elliptical corners.

box-shadow & outline
shadow & outline
div { box-shadow: 0 10px 30px rgba(0,0,0,0.5); outline: 2px solid rgba(99,102,241,0.25); }
Details

Use multiple shadows for layered depth. outline is useful for focus states and doesn't change element size.

Positioning & Layout

How elements are laid out on the page — flow, stacking, and advanced clipping.

display
display: inline-block;
div { display: inline-block; }
Details

display value determines layout model. Use display:flex/grid for modern layouts. display:none hides an element from layout and accessibility tree unless aria-hidden used carefully.

position, top/right/bottom/left
relative / absolute / sticky
.box { position: relative; } .box .child { position: absolute; top: 10px; right: 10px; }
Details

position:sticky is useful for headers that stick within a scroll container. fixed positions relative to viewport.

z-index
z-index
.front { z-index: 10; position: relative; }
Details

z-index works on positioned elements (position other than static). New stacking contexts can be created by transforms, opacity less than 1, and certain properties.

float & clear
float: left;
.img { float: left; margin-right: 12px; } .clearfix::after { content: ''; display: table; clear: both; }
Details

Float is legacy for layouts; prefer flex and grid. Use clearfix to clear floats in a container.

overflow & clip-path
clip-path + overflow
.card { overflow: auto; clip-path: polygon(...); }
Details

clip-path is great for creative shapes; use SVG clipPath for complex shapes. overflow:auto adds scrollbars when needed.

Grid

Two-dimensional layout for rows and columns. Powerful for page-level layouts and complex components.

display: grid
1
2
3
.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 8px; }
Details

Grid excels at 2D layout (rows + columns). Use grid-template-areas for named placement.

grid-template / gap
left
main
.layout { grid-template-columns: 120px 1fr; gap: 12px; }
Details

Use auto-fill / auto-fit with minmax() to create responsive grid columns.

grid-column / grid-row

Advanced Visual Effects

Filters, blending, masks, and visual layers that give modern depth and polish.

opacity
opacity: 0.85
div { opacity: 0.85; }
Details

Opacity affects the element and its children. For backdrop blurs, use rgba backgrounds and backdrop-filter.

filter
grayscale(100%)
img { filter: grayscale(100%) blur(2px) contrast(1.1); }
Details

Multiple filter functions can be chained. Filters can be GPU-accelerated in many browsers.

backdrop-filter
backdrop-filter: blur(6px)
.glass { background: rgba(255,255,255,0.06); backdrop-filter: blur(6px); }
Details

Not supported in older browsers. Often used with semi-transparent backgrounds to create glassmorphism.

mix-blend-mode
mix-blend-mode: screen
.badge { mix-blend-mode: screen; }
Details

Great for layered UI and hero sections. Can create surprising results; test on multiple backgrounds.

mask-image
mask-image gradient
.fade { mask-image: linear-gradient(to right, transparent, black); }
Details

mask-image hides parts of an element. Use SVG masks for precise shapes and alpha control.

object-fit & object-position
img { width: 100%; height: 200px; object-fit: cover; object-position: center; }
Details

object-fit: cover preserves aspect ratio while filling the container. Use for responsive image containers.

Transformations & Animations

Transforms let you move and rotate elements; transitions and animations add motion and life.

transform & transform-origin
rotate(-6deg) scale(1.02)
.logo { transform: rotate(-6deg) scale(1.02); transform-origin: center; }
Details

Transforms do not affect document flow; they create new stacking contexts in some cases.

transition
hover to animate
.btn { transition: transform 300ms ease; } .btn:hover { transform: scale(1.06); }
Details

Transition only animates changes; use for hover/active state transitions for smoother UI.

animation & @keyframes
@keyframes bounce { to { transform: translateY(-10px); } } .dot { animation: bounce 900ms infinite alternate; }
Details

Use animation-duration, iteration-count, timing-function, delay, and direction. Prefer reduced-motion media query for accessibility.

perspective
.scene { perspective: 600px; } .card { transform: rotateY(25deg); }
Details

Perspective gives depth for 3D transforms; perspective-origin controls the vanishing point.

Lists & Tables

Styling lists and tables for readable content and data display.

list-style
  • disc
  • circle
ul { list-style-type: disc; list-style-position: inside; }
Details

Use custom list images with list-style-image or pseudo-elements for richer designs.

table layout
AB
table { border-collapse: collapse; border-spacing: 0; }
Details

Tables are for tabular data — avoid using them for layout. Use aria attributes for accessibility.

Responsive Design & Media Queries

Make layouts adapt using queries, responsive units, and modern functions.

@media
@media (max-width: 600px)
@media (max-width: 600px) { .nav { display: none; } }
Details

Use mobile-first approach (min-width) for easier scaling. Use feature queries (@supports) for progressive enhancement.

clamp(), min(), max()
font-size: clamp(1rem, 2.5vw, 1.5rem)
h1 { font-size: clamp(1rem, 2.5vw, 1.5rem); }
Details

clamp() helps create fluid yet constrained sizes without media queries. Useful for typography.

@supports
@supports(display: grid)
@supports (display: grid) { .layout { display: grid; } }
Details

Useful for progressive enhancement: only apply modern features when the browser supports them.

Variables & calc()

Custom properties let you create themeable, dynamic CSS. calc() performs math inside CSS.

CSS Custom Properties
--accent: #06b6d4;
:root { --accent: #06b6d4; } .btn { background: var(--accent); }
Details

Custom properties are dynamic and can be updated at runtime (JS or via media queries). Great for themes.

calc() & units
width: calc(100% - 48px)
.main { width: calc(100% - 48px); }
Details

calc() lets you combine px, %, vw, rem etc. Useful for precise responsive layouts.
