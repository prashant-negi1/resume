Text & Fonts

Typography, spacing, and casing

font-family text

Set a safe, modern stack that falls back gracefully.

body{
  font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
               Helvetica, Arial, "Apple Color Emoji", "Segoe UI Emoji";
}

fluid type with clamp() text

Scale font size smoothly between min and max.

h1{ font-size: clamp(1.5rem, 1rem + 3vw, 3rem); }

line-height & letter-spacing text

Improve readability for paragraphs and headings.

p{ line-height: 1.7; }
h1{ letter-spacing: .3px; }

text-transform & text-overflow text

Uppercase titles and ellipsis for overflow.

.kicker{ text-transform: uppercase; letter-spacing: .15em; }
.title{ white-space: nowrap; overflow: hidden; text-overflow: ellipsis; }

Colors & Backgrounds

Gradients, images, and clipping

linear-gradient background background

Quick hero background with two brand colors.

.hero{ background:
  linear-gradient(135deg, #00adb5, #4ade80);
}

background-size, position, repeat background

Control image placement and scaling.

.banner{
  background-image: url("hero.jpg");
  background-size: cover;     /* or contain */
  background-position: center;
  background-repeat: no-repeat;
}

gradient text with background-clip effect

Create eye‑catching titles without images.

.gradient-text{
  background: linear-gradient(90deg, #00adb5, #4ade80);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
}

Layout (Flexbox & Grid)

Modern 1D & 2D layout systems

flexbox basics layout

Horizontal center with spacing and wrapping.

.toolbar{
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1rem; flex-wrap: wrap;
}

flex alignment tricks layout

Center anything perfectly.

.center{
  display:flex; min-height: 50vh;
  justify-content:center; align-items:center;
}

grid columns & gap layout

Responsive grid that switches at 700px.

.grid{ display:grid; grid-template-columns: 1fr; gap: 1rem; }
@media(min-width:700px){
  .grid{ grid-template-columns: repeat(3, 1fr); }
}

place-items & minmax() layout

Auto-fit smart columns that don’t shrink too small.

.gallery{
  display:grid; grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  place-items:center; gap:1rem;
}

Positioning

Sticky, fixed, absolute

position: sticky position

Make headers stick within their parent container.

.section-title{ position: sticky; top: 0; background: white; z-index: 10; }

z-index layering position

Control stacking order of overlapping elements.

.modal{ position: fixed; inset: 0; z-index: 50; }
.header{ position: sticky; top: 0; z-index: 40; }

Transitions & Animations

Microinteractions that feel good

transition shorthand motion

Smooth hover effects in one line.

.btn{ transition: transform .15s ease, box-shadow .2s ease; }
.btn:hover{ transform: translateY(-2px); box-shadow: 0 8px 24px rgba(0,0,0,.25); }

@keyframes + animation motion

Pulse a focus ring or loading dot.

@keyframes pulse{ from{ box-shadow:0 0 0 0 rgba(0,173,181,.5) } to{ box-shadow:0 0 0 14px rgba(0,173,181,0) } }
.badge{ animation: pulse 1.5s infinite; }

Borders & Shadows

Depth and emphasis

box-shadow & border-radius ui

Soft, modern card style.

.card{ border-radius: 16px; box-shadow: 0 10px 30px rgba(2,6,23,.35); }

outline & focus-visible a11y

Accessible focus style for keyboard users.

:focus-visible{ outline: 2px solid #00adb5; outline-offset: 2px; }

Extras

Object-fit, overflow, filters

object-fit for images media

Crop images responsively without distortion.

.thumb{ width:100%; height:220px; object-fit:cover; border-radius:12px; }

overflow & scroll snapping scroll

Horizontal galleries with smooth snaps.

.row{ display:flex; gap:1rem; overflow:auto; scroll-snap-type:x mandatory; }
.row > *{ scroll-snap-align: start; }
